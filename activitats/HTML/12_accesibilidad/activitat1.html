Cosas que podrian mejorar las paginas web

  • Añadir aria-label a iconos y botones ambiguos
  • Usar imágenes con alt descriptivo
  • Asegurar un orden lógico de tabulación
  • Mantener una jerarquía de encabezados adecuada
  • Evitar el texto de bajo contraste
  • No depender solo del color para transmitir información
  • Proveer mensajes visibles y auditivos en elementos interactivos
